Resumo – Enumeração Web no domínio AetherPharma

O desafio começou com a enumeração web do domínio principal aetherpharma.sy. Durante os primeiros testes, qualquer tentativa de acessar páginas como /feedback, /staff ou enviar dados pelo formulário retornava 404 Not Found, indicando que o problema não estava em paths comuns.

🔍 Identificação de Virtual Hosts

A dica da plataforma mencionava explicitamente a existência de dois domínios:

feedback

staff

Isso indicava o uso de Virtual Hosts (vhosts) no Apache, e não diretórios no domínio principal.

Ao tentar acessar diretamente:

feedback.aetherpharma.sy
staff.aetherpharma.sy


o DNS não resolvia os nomes, o que reforçou a hipótese de vhosts sem DNS público, um padrão clássico em CTFs.

🌐 Forçando o Host Header

Após descobrir o IP do servidor (34.205.41.50), os vhosts foram testados manualmente usando o header Host:

curl -H "Host: feedback.aetherpharma.sy" http://34.205.41.50
curl -H "Host: staff.aetherpharma.sy" http://34.205.41.50


Ambos retornaram HTTP 200 OK, confirmando a existência dos dois virtual hosts:

feedback.aetherpharma.sy → página HTML estática

staff.aetherpharma.sy → aplicação PHP com sessão (PHPSESSID)

🧠 Exploração do domínio feedback

Seguindo a dica do desafio, o foco inicial foi o domínio feedback.
Ao acessar o conteúdo completo da página e analisar o código-fonte HTML, foi encontrado um comentário escondido no final do documento:

<!--Solyd{!Th3F1Rs7oFM4NYflAGs!!H4V3Fun!}-->


Esse comentário continha diretamente a flag do desafio, sem necessidade de exploração ativa do formulário.
